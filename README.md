# 개요 :
본 프로젝트는 '엘포박스' 기업과의 협력을 통해 진행된 프로젝트로, <br>
기업에서 지정한 주요 포인트를 바탕으로 기획팀이 세운 게임기획에 따라 4개의 미니게임과 초기 진입 마을을 개발하였습니다.<br>

### [ 기업 주요 요구사항 ] 
- 스크린: 대형 55인치 활용
- 게임 타입: 터치 미니게임
- 타겟 사용자: 4세 ~ 9세 어린이
- 플랫폼: Android 10.0 (API level 29) 호환 

# 담당 업무 :
- SketchUP 미니게임 구현
- 마을 랭킹 적용 (몽고디비 아틀란스 데이터베이스 활용)
