# 개요 :
본 프로젝트는 '엘포박스' 기업과의 협력을 통해 진행된 프로젝트로, 기업에서 지정한 주요 포인트를 바탕으로 기획팀이 세운 방향에 따라 4개의 미니게임과 초기 진입 마을 개발 작업을 수행하였습니다.
- 기업 키 포인트 
 - 대형 55인치 스크린 활용
 - 터치 미니게임
 - 타겟 사용자: 4세 ~ 9세 어린이
 - Android 10.0 (API level 29) 호환
